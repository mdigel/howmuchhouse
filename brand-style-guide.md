This project mimics the Uber design system (Uber Base). 
Core Principles:
1. **Bold & High Contrast**: Use absolute Black (#000000) and White (#FFFFFF) as primary colors. 
2. **Typography-Driven**: Use "Inter" (as proxy for Uber Move). Headings should be bold, tight, and commanding.
3. **Minimalist Layouts**: Extensive whitespace. Clean grids. Sharp corners or subtle rounding (8px/0.5rem).
4. **Iconography**: Simple, stroke-based or solid glyphs.
5. **Brand Color**: Use Uber 
Blue (#276EF1) only for accents, links, and key indicators. NOT for main buttons (Main buttons are Black).
6. **Mobile-First**: Design should feel like a native app experience.

Visual Style:
- Buttons: Solid Black with White text (Primary). Light Gray (#F3F3F3) with Black text (Secondary).
- Inputs: Gray backgrounds (#F3F3F3) or clean borders.
- Shadows: Subtle, functional shadows.
